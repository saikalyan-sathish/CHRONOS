# Contributing to Chronos

Thank you for your interest in contributing to Chronos! This document provides guidelines for contributing to this project.

## üîí Contribution Policy

This is a **personal project** and contributions are accepted **by invitation only**.

### How to Contribute

1. **Do NOT** submit unsolicited pull requests - they will be closed without review
2. If you'd like to contribute, please [open an issue](../../issues/new) first to discuss your proposed changes
3. Wait for approval from the maintainer before starting any work
4. Only after receiving explicit permission should you fork and submit a PR

### Reporting Bugs

Bug reports are welcome! Please:
1. Check if the issue already exists
2. Open a new issue with a clear description
3. Include steps to reproduce the bug
4. Mention your browser and OS version

### Feature Requests

Feature suggestions are appreciated! Please:
1. Open an issue describing the feature
2. Explain why it would be useful
3. Wait for maintainer feedback before implementation
